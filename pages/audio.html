O elemento HTML

O elemento HTML <audio>

Para reproduzir um arquivo de áudio em HTML, use o <audio>elemento:

            <audio controls>
                <source src="audio/sci-fi-sound-effect-designed-circuits-hum-24-200825.mp3" type="audio/mpeg">
              </audio>

Audio HTML Como Funciona

O controls atributo adiciona controles de áudio, como reproduzir, pausar e volume.

O elemento <source> permite que você especifique arquivos de áudio alternativos que o navegador pode escolher. O navegador usará o primeiro formato reconhecido.

O texto entre as tags <audio> e </audio> só será exibido em navegadores que não suportam o <audio> elemento.

HTML <audio> Reprodução automática

Para iniciar um arquivo de áudio automaticamente, use o autoplay atributo:

        <audio controls autoplay>
            <source src="audio/sci-fi-sound-effect-designed-circuits-hum-24-200825.mp3" type="audio/mpeg">
        </audio>

Para iniciar um arquivo de áudio automaticamente, use o autoplay atributo. Para reprodução automática sem som, adicione o muted atributo:

        <audio controls autoplay muted>
            <source src="audio/sci-fi-sound-effect-designed-circuits-hum-24-200825.mp3" type="audio/mpeg">
          </audio>
