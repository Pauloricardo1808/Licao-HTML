Elemento <form>

O elemento HTML <form> é usado para criar um formulário HTML para entrada do usuário:

. Elementos do formulário .

O elemento <label>

Observe o uso do <p>elemento no exemplo acima.

O <label>elemento define um rótulo para muitos elementos do formulário.

O <label>elemento é útil para usuários de leitores de tela, porque o leitor de tela lerá o rótulo em voz alta quando o usuário focar no elemento de entrada.

O <label>elemento também ajuda usuários que têm dificuldade em clicar em regiões muito pequenas (como botões de opção ou caixas de seleção) - porque quando o usuário clica no texto dentro do <p>elemento, ele alterna o botão de opção/caixa de seleção.

O foratributo da <label>tag deve ser igual ao idatributo do <input> elemento para uni-los.

O elemento <input>

O elemento HTML <input> é o elemento de formulário mais usado.

Um <input> elemento pode ser exibido de várias maneiras, dependendo do type atributo.

aqui estão alguns exemplos:

<input type="texto">

Exibe um campo de entrada de texto de linha única

<input type="text" placeholder="Digite seu texto aqui">

<input type="rádio">

Exibe um botão de opção (para selecionar uma das muitas opções)

<input type="radio" id="option1" name="option" value="option1"> <label for="option1">Opção 1</label>
<input type="radio" id="option2" name="option" value="option2"> <label for="option2">Opção 2</label>



<input type="caixa de seleção">

Exibe uma caixa de seleção (para selecionar zero ou mais dentre muitas opções)

<input type="checkbox" id="option3" name="option3" value="option3"> <label for="option3">Opção 3</label>
<input type="checkbox" id="option4" name="option4" value="option4"> <label for="option4">Opção 4</label>



<input type="enviar">

Exibe um botão enviar (para enviar o formulário)

<input type="submit" value="Enviar">

<input type="botão">

Exibe um botão clicável

<input type="button" value="Clique Aqui">
