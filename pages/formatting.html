HTML Elementos <i> e <em>

O elemento HTML <i> define uma parte do texto em uma voz ou humor alternativo. O conteúdo interno geralmente é exibido em itálico.

Dica: A tag <i> é frequentemente usada para indicar um termo técnico, uma frase de outro idioma, um pensamento, um nome de navio, etc.

Exemplo: <i>Este texto está em itálico</i>

O elemento HTML <em> define texto enfatizado. O conteúdo interno geralmente é exibido em itálico.

Dica: Um leitor de tela pronunciará as palavras em <em> com ênfase, usando estresse verbal.

Exemplo: <em>Este texto está enfatizado</em>

HTML Elemento <small>

O elemento HTML <small> define texto menor:

Exemplo: <small>Este é um texto menor.</small>

HTML Elemento <mark>

O elemento HTML <mark> define texto que deve ser marcado ou destacado:

Exemplo: <p>Não se esqueça de comprar <mark>leite</mark> hoje.</p>

HTML Elemento <del>

O elemento HTML <del> define texto que foi deletado de um documento. Os navegadores geralmente riscam uma linha através do texto deletado:

Exemplo: <p>Minha cor favorita é <del>azul</del> vermelho.</p>

HTML Elemento <ins>

O elemento HTML <ins> define um texto que foi inserido em um documento. Os navegadores geralmente sublinham o texto inserido:

Exemplo: <p>Minha cor favorita é <del>azul</del> <ins>vermelho</ins>.</p>

HTML Elemento <sub>

O elemento HTML <sub> define texto em subscrito. O texto em subscrito aparece metade de um caractere abaixo da linha normal e às vezes é renderizado em uma fonte menor. O texto em subscrito pode ser usado para fórmulas químicas, como H2O:

Exemplo: <p>Este é um texto <sub>em subscrito</sub>.</p>

HTML Elemento <sup>

O elemento HTML <sup> define texto em sobrescrito. O texto em sobrescrito aparece metade de um caractere acima da linha normal e às vezes é renderizado em uma fonte menor. O texto em sobrescrito pode ser usado para notas de rodapé, como WWW<sup>[1]</sup>:

Exemplo: <p>Este é um texto <sup>em sobrescrito</sup>.</p>
