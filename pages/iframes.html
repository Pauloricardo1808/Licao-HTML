Use os atributos height e width para especificar o tamanho do iframe.

Exemplo:

<iframe src="attributes.html" height="200" width="300" title="Iframe Example"></iframe>

Ou você pode adicionar o atributo style e usar o CSS height e width como propriedades:

Exemplo:

<iframe src="attributes.html" style="height:200px;width:300px;" title="Iframe Example"></iframe>

Para remover a borda, adicione o atributo style e use a propriedade CSS border:

Exemplo:

<iframe src="attributes.html" style="border:none;" title="Iframe Example"></iframe>

Com CSS, você também pode alterar o tamanho, estilo e cor da borda do iframe:

Exemplo:

<iframe src="attributes.html" style="border:2px solid red;" title="Iframe Example"></iframe>

Um iframe pode ser usado como quadro de destino para um link.

O target atributo do link deve referir-se ao name atributo do iframe:

Exemplo:

<iframe src="attributes.html" name="iframe_a" title="Iframe Example"></iframe>
