Tag de Comentário HTML

Você pode adicionar comentários ao seu código HTML usando a seguinte sintaxe:

<!-- Escreva seus comentários aqui -->

Observe que há um ponto de exclamação (!) na tag de abertura, mas não na tag de fechamento.

Adicionar Comentários

Com comentários, você pode colocar notificações e lembretes em seu código HTML:

<!-- Este é um comentário -->

Este é um parágrafo.

<!-- Lembre-se de adicionar mais informações aqui -->

Ocultar Conteúdo

Comentários podem ser usados para ocultar conteúdo.

Isso pode ser útil se você precisar ocultar conteúdo temporariamente:

Este é um parágrafo.

<!-- Este é outro parágrafo -->

Este é outro parágrafo também.

Ocultar Conteúdo em Linha

Comentários podem ser usados para ocultar partes no meio do código HTML.

Ocultar uma parte de um parágrafo:

Este <!-- ótimo texto --> é um parágrafo.
