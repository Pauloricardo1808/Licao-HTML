Gráficos de tela HTML

O elemento HTML <canvas> é usado para desenhar gráficos em uma página da web.

O gráfico à esquerda é criado com <canvas>. Ele mostra quatro elementos: um retângulo vermelho, um retângulo gradiente, um retângulo multicolorido e um texto multicolorido.

O que é tela HTML?

O elemento HTML <canvas> é usado para desenhar gráficos dinamicamente, via JavaScript.

O <canvas> elemento é apenas um contêiner para gráficos. Você deve usar JavaScript para realmente desenhar os gráficos.

O Canvas possui vários métodos para desenhar caminhos, caixas, círculos, texto e adicionar imagens.

Canvas é compatível com todos os principais navegadores.

Nota: Sempre especifique um id atributo (a ser referenciado em um script) e um atributo width e height para definir o tamanho da tela. Para adicionar uma borda, use o style atributo.
