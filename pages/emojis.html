O que são Emojis?

Emojis parecem imagens ou ícones, mas não são.

Eles são letras (caracteres) do conjunto de caracteres UTF-8 (Unicode).

O Atributo de charset HTML

Para exibir uma página HTML corretamente, um navegador da web deve saber o conjunto de caracteres usado na página.

Isso é especificado na tag <meta>:

<meta charset="UTF-8">

Caracteres UTF-8

Muitos caracteres UTF-8 não podem ser digitados em um teclado, mas sempre podem ser exibidos usando números (chamados de números de entidade):

A é 65

B é 66

C é 67
