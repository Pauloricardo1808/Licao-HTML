Cabeçalhos HTML são títulos ou subtítulos que você deseja exibir em uma página da web.

<h1>Cabeçalho 1</h1>

<h2>Cabeçalho 2</h2>

<h3>Cabeçalho 3</h3>

<h4>Cabeçalho 4</h4>

<h5>Cabeçalho 5</h5>

<h6>Cabeçalho 6</h6>

Cabeçalhos HTML

Os cabeçalhos HTML são definidos com as tags <h1> a <h6>.

<h1> define o cabeçalho mais importante. <h6> define o cabeçalho menos importante.

Exemplo

Cabeçalho 1

Cabeçalho 2

Cabeçalho 3

Cabeçalho 4

Cabeçalho 5
Cabeçalh1o 6

Observação: Navegadores adicionam automaticamente algum espaço em branco (uma margem) antes e depois de um cabeçalho.

Cabeçalhos São Importantes

Mecanismos de busca usam os cabeçalhos para indexar a estrutura e o conteúdo das suas páginas da web.

Usuários geralmente percorrem uma página pelos seus cabeçalhos. É importante usar cabeçalhos para mostrar a estrutura do documento.

<h1> deve ser usado para cabeçalhos principais, seguido por <h2>, e assim por diante.

Observação: Use cabeçalhos HTML apenas para cabeçalhos. Não use cabeçalhos para tornar o texto GRANDE ou em negrito.

Cabeçalhos Maiores

Cada cabeçalho HTML tem um tamanho padrão. No entanto, você pode especificar o tamanho para qualquer cabeçalho com o atributo style, usando a propriedade CSS font-size:

Exemplo

<h1 style="font-size:60px;">Cabeçalho 1</h1>
